Task Description
Write a program to read a number x
between 100
and 999
, then print the first, the second and the third digit of x
. Every digit is in one line.
Input Format
There is one line in the input. The line has a number x
.
Output Format
There are three lines in the output. The first line has the first digit of x
; the second line has the second digit, then the third line has the third digit.
Sample Input
123
Sample Output
1
2
3
